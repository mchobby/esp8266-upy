# Bibliothèque stat

La bibliothèque [stat.py](lib/stat.py) contient la classe `Statistic` utilisé pour calculer le  minima, maxima, mean, variance, 
standard deviation d'une série de valeurs poussées (une par une) dans la classe.

## note:

Basé sur les travaux répertoriés dans le fil de discussion "[Updated Statistic library](https://forum.arduino.cc/t/updated-statistics-library/295881/3)" pour Arduino
