# stat library

The [stat.py](lib/stat.py) library contains the `Statistic` class used to calculate  minima, maxima, mean, variance, 
standard deviation for a serie of value pushed (one by one) into the class.

## Note:

This library is based on the works mentionned in the thread "[Updated Statistics library](https://forum.arduino.cc/t/updated-statistics-library/295881/3)"
